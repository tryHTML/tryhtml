## Installing

Using npm:

```md
$ npm install -g tryhtml
```
or

```md
$ npm install --global tryhtml
```
View on [npm](https://www.npmjs.com/package/tryhtml).
Using Yarn

```md
$ yarn add -dev tryhtml
```
View on [Yarn](https://yarnpkg.com/en/package/tryhtml).
